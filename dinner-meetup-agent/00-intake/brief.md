## The Idea

**产品名:** 「今天整点啥」(What's the Plan)
**Slogan:** 少废话，直接出发

**Problem:** 熟人聚会最大痛点不是"没地方玩"，而是：
1. 群里问一圈没人出主意，最后不了了之
2. 每个人喜好不同，协调成本极高
3. 有人出了方案，又反复被改，组织者心力交瘁

**Solution:** 用 AI 替代"群里那个最操心的人"：
- 组织者创建聚会 → 生成房间码+链接，发到微信群
- 参与者各自通过 H5 与 AI 对话（4-5 轮）收集偏好：氛围倾向、饮食禁忌、预算、位置、活动排除
- AI 偏好融合算法（硬性禁忌优先过滤 → 加权多数决策 → 中位数预算 → 主方案+方案B）
- 输出一站式聚会方案，支持动态修改

**Trigger:** 个人痛点 + 对多 agent 协作架构的技术兴趣

**Existing work:** PRD v0.1 已完成，包含完整的用户旅程、功能模块、页面清单和风险评估

## Product Scope (from PRD v0.1)

- **平台:** Web App（H5，支持微信内打开跳转）
- **视觉:** 橙+白撞色，圆润粗体，卡片式布局，高饱和，互联网梗文案
- **数据源:** 高德地图/腾讯地图 POI API + 兜底 JSON
- **P1 页面:** 8 个（引导页、身份选择、创建聚会、邀请页、等待页、偏好对话、填写完成、方案展示）
- **核心功能:** 创建聚会、偏好收集（4-5 轮 AI 对话）、偏好融合算法、方案生成与展示、动态修改
- **范围外:** 账号体系、历史记录、真实优惠券、多城市、地图动线可视化

## The Market

- **目标用户:** 熟人聚会场景（朋友、家人、同事小团）
- **地理范围:** 演示版固定深圳南山区
- **现有替代方案:** 群聊商量、大众点评搜索、简单投票工具
- **直接竞品:** 未系统调研

## Constraints

- **技术:** 高德/腾讯地图 POI API 为主数据源
- **平台:** H5 Web App，微信内可打开
- **演示定位:** 深圳南山区
- **成功标准:** 完整走通「创建聚会 → 填偏好 → 生成方案 → 修改方案」全流程
